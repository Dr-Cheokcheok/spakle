## :bookmark_tabs: 프로젝트 세부 계획
+ 회원관리 서비스
  - 회원가입
  - 회원탈퇴
  - 회원정보수정
  - 마이페이지
  - 로그인/로그아웃
+ 결제 서비스
  - 장바구니에 상품 등록 및 삭제
  - 결제 시 장바구니 목록보기
  - 제품 결제
+ 관리자 서비스
  - 상품 등록 및 삭제
  - 고객들이 결제한 상품목록 조회

  - 자주 찾는 상품 찜 기능
